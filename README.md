# DriveSyncPy

**Still in development. Not ready for productive use.**

A Python Linux tool that synchronises Google Drive content with your local machines. It detects changes in the file system locally and remotely and automatically synchronises with the other system. Based on Googleâ€™s Drive API.


### Development
Optionally, create a virtual environment. Then install the dependencies.

```bash
git clone https://github.com/lawrencebenson/drivesyncpy.git && cd drivesyncpy
virtualenv . -p python3 && source bin/activate
pip3 install -r requirements.txt
```
